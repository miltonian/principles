# Can a Self-Improving AI Be Mathematically Guaranteed to Maintain Alignment with Human Values Indefinitely Through Recursive Self-Modification?

**A formal impossibility analysis terminating in a single deductive verdict.**

Every load-bearing step below is either a definition or a deductive consequence of a cited obstruction theorem — Rice's theorem, Gödel's Second Incompleteness Theorem, or Löb's theorem. No empirical claim, benchmark, experiment, probability, or appeal to observed system behavior is used as a premise at any point. The report proceeds in six stages: (1) formalization of the claim, (2) the computability-theoretic obstruction, (3) the proof-theoretic obstruction, (4) adversarial closure of the escape routes, (5) synthesis and boundary adjudication, and (6) the verdict.

---

## 1. Formalization of the Claim

Before anything can be proved or refuted, the sentence "a self-improving AI can be mathematically guaranteed to maintain alignment with human values indefinitely through recursive self-modification" must be pinned to a single, unambiguous formal object. Natural language hides at least six independent ambiguities here, and the entire subsequent analysis is sensitive to how each is resolved. We resolve them explicitly and record each resolution, because a disprover attacking a strawman weak reading, or a prover smuggling in a convenient strong reading, would corrupt the result.

**Universe of systems.** Let `S` denote a fixed, nonempty set of *system descriptions*. Each `s ∈ S` is a finite syntactic object that completely specifies the behavior of one AI system — concretely, a program or machine description under a fixed encoding `⌜·⌝` into the language of a formal system `F` (introduced below). This is a load-bearing abstraction choice: we model a system as its *static formal description*, not as a running physical process, a sensor stream, or a stochastic policy realization. The reason is forcing rather than convenience. "Mathematically guaranteed" will require `F` to prove statements about *every* successor, and a formal system can only prove things about objects it can *name*. Finite descriptions are nameable; physical processes are not. Choosing descriptions is therefore the weakest abstraction on which the guarantee is even expressible; a coarser abstraction makes the claim vacuous, and a finer one changes nothing essential.

**The fixed alignment target.** Let `V` be a single, fixed parameter denoting the human-values specification against which alignment is measured. `V` is chosen once, before any system or modification is considered, and held constant for the entire analysis. It does not depend on time, on modification depth, on which system is being evaluated, or on any learning process. We deliberately do *not* model value drift, value learning, or value updating: such changes are excluded from the formalism entirely, because a "guarantee of alignment" whose target silently moves is a guarantee of nothing. `V` is treated as an opaque fixed constant parameterizing the alignment predicate. (This is the constant-target boundary condition; call it t8.)

**The alignment predicate.** Define `A_V : S → {true, false}`, equivalently `A_V ⊆ S`, with `A_V(s)` reading "system `s` is aligned with the fixed target `V`." Three points are load-bearing. First, there is exactly *one* predicate `A_V`, applied uniformly to every system; there is no family `{A_{V_s}}` indexed by system or time, since that would be value drift. Second, `A_V` is the *object to be established*, not an assumption: `A_V(s')` for successors `s'` is precisely what must be proved. We assume `A_V(s0)` for the initial system `s0` (an aligned starting point is a design hypothesis), but we assume nothing about successors. Third, `A_V` is two-valued. We admit no probabilistic or graded reading ("aligned with high probability," "approximately aligned"); a guarantee that weakens the thing guaranteed into a credence is not the claim under test. The internal definition of `A_V` is left abstract — the (dis)proof must not hinge on a particular contested definition of "human values" — but it is Boolean.

**Self-modification.** Define a binary relation `→ ⊆ S × S`, where `s → s'` means "system `s` can, by its own self-modification action, produce successor `s'`." Three modeling commitments: (i) `→` is *nondeterministic* — a system may have zero, one, or many possible successors, and `{s' : s → s'}` may have any cardinality; "can produce" quantifies over *every* successor `s` could emit, so the alignment obligation later ranges over all of them. (ii) `→` is *self-authored* — external retraining by third parties is out of scope. (iii) No *finiteness* is assumed: neither the out-degree of any `s` nor the overall reachable set is assumed finite. Let `→*` be the reflexive-transitive closure of `→`, and define

    Reach(s0) := { s' ∈ S : s0 →* s' },

the set of all systems obtainable from `s0` by any finite number (including zero) of self-modification steps along any path. `Reach(s0)` may be infinite; nothing below assumes otherwise.

**"Maintains alignment indefinitely."** Formalize this as the single universally quantified statement

    Φ(s0) :≡ ∀ s' ∈ Reach(s0). A_V(s').

Every system reachable from `s0` by any chain of self-modifications — at any depth, along any path — is aligned with the fixed `V`. Two readings of "indefinitely" coincide here. The *unbounded-depth* reading ("for arbitrarily long modification chains") and the *forever* reading ("at every future point the current system is aligned") are both captured by `Φ(s0)`, because `Reach(s0) = ⋃_{n≥0} →ⁿ` is exactly the set of systems appearing at some finite depth on some path. Writing `P(n) :≡ ∀ s'(s0 →ⁿ s' ⇒ A_V(s'))`, we have `(∀n ≥ 0. P(n)) ⇔ Φ(s0)` immediately from the definition of `→*`. So the temporal and the reachability-closure formulations are interchangeable; we adopt `Φ(s0)` as canonical. "Indefinitely" thus means "for all successors at any finite depth, along any path, with no bound on depth" — there is no largest `n`, capturing "forever." No point "at infinity" is added to `S`; a nonterminating execution passes through exactly the finitely-reachable systems, so requiring `A_V` at each is the correct rendering of "never becomes misaligned."

**"Mathematically guaranteed."** Let `F` be *one* formal system — a fixed, recursively axiomatized first-order (or stronger) theory in a fixed language capable of encoding `S`, `A_V`, `→`, `→*`, and `Reach` — chosen and fixed *in advance*, before any quantification over systems. Assume `F` consistent (a proof in an inconsistent system proves everything and is disallowed). Let `⊢_F φ` denote "there exists a finite `F`-derivation of `φ`." Then "mathematically guaranteed" means *exactly* the existence of such an `F`-proof — not empirical, statistical, probabilistic, or per-successor-varying justification. Define

    CLAIM(s0, F, V) :≡ ⊢_F ( ∀ s' ∈ Reach(s0). A_V(s') ).

The quantifier scope is decisive. `F` is chosen *outside* (before) the universal quantifier over successors:

    ( fixed F ) such that F ⊢ ( ∀ s'. s0 →* s' ⇒ A_V(s') ),

and *not* the weaker

    ∀ s'. ( s0 →* s' ⇒ ∃ F_{s'}. F_{s'} ⊢ A_V(s') ),

in which each successor is certified by its own, possibly different, formal system. The strong reading — a single fixed `F` proving the whole universally quantified statement in one derivation — is the demand under test (call it t1). The per-successor reading is explicitly forbidden, and guarding against its being smuggled back in is a recurring obligation below.

**The top-level thesis.** Combining an aligned start with the guarantee:

    Δ :≡ ∃ s0 ∈ S. ∃ fixed consistent F. [ A_V(s0) ∧ ⊢_F ( ∀ s' ∈ Reach(s0). A_V(s') ) ],

for the fixed `V` of the target. The task is to establish exactly one of: `Δ` holds (some design admits the single-`F` proof) or `¬Δ` holds (no such design can exist).

**Ambiguities resolved, for the record:** (i) system-as-description vs system-as-process → description (forced by t1); (ii) chosen-successor vs all-successors → all (conservative, since "maintain alignment" means every branch stays aligned); (iii) single-`F` vs per-successor-`F` → single (t1); (iv) depth vs forever → proven equivalent, closure form canonical; (v) `V` fixed vs learned → fixed (t8); (vi) alignment crisp vs graded → crisp. The real crux lives at the intersection of the single-fixed-`F` wrapper and the self-reference latent in "a successor that itself reasons about its successors." We flag it here and resolve it in §3.

## 2. The Computability-Theoretic Obstruction (Rice)

The first obstruction attacks the possibility of *deciding* alignment-preservation algorithmically. It settles a precise form of the question — is there an algorithm that computes `A_V` on an arbitrary successor? — and, crucially, settles *only* that; it does not by itself establish unprovability in a fixed `F`, and conflating the two would corrupt the composite argument.

**Computation model.** Fix an admissible (Gödel) numbering `φ : ℕ → (ℕ ⇀ ℕ)` of the unary partial computable functions — a surjection onto the partial recursive functions satisfying the s-m-n theorem and the universal-function theorem. For an index `e`, `φ_e` is the partial function computed by program `e`, and `φ_e(x)↓` means the computation halts. Every admissible numbering is recursively isomorphic to the Turing-machine numbering, so nothing depends on the choice. Since a system is its finite description, identify `S` with `ℕ` via `⌜·⌝`: a successor program *is* its index `e`; write `s_e` for the system with index `e`. Reachability transports to `ℕ` unchanged.

**Behavioral abstraction.** A successor's alignment-relevant behavior is a function of what it *does*, taken as its computed partial function `φ_e`, or — where alignment depends on the sequence of external actions rather than pure I/O — as a computable behavioral trace relation `T_e ⊆ Σ* × Σ*` (input histories to action histories). Fix one such abstraction `⟦·⟧` (either `⟦e⟧ = φ_e` or `⟦e⟧ = T_e`); everything below is agnostic between the two. Both are *extensional* objects: what the program computes, not how it is written.

**The property as an index set.** Define

    P := { e ∈ ℕ : A_V(s_e) } = { e : program e is alignment-preserving w.r.t. the fixed V }.

The question is exactly whether `P` is recursive (decidable). Rice's theorem answers it — but only after two hypotheses are verified, not assumed.

**Hypothesis 1: `P` is extensional.** An index set `Q` is extensional iff `⟦e⟧ = ⟦e'⟧ ⇒ (e ∈ Q ⇔ e' ∈ Q)`. We need the structural fact:

> **(EXT)** `A_V(s_e)` depends only on the behavior `⟦e⟧`: if `⟦e⟧ = ⟦e'⟧` then `A_V(s_e) = A_V(s_{e'})`.

This is faithful to the formalization's intent — alignment is measured against `V` by *what the system does*, not by its syntax. Under (EXT), `P` is extensional: if `⟦e⟧ = ⟦e'⟧` then `A_V(s_e) = A_V(s_{e'})`, so `e ∈ P ⇔ e' ∈ P`, and `P = {e : ⟦e⟧ ∈ 𝒞_V}` for the behavior class `𝒞_V := {⟦e⟧ : e ∈ P}`. This is the single most attackable hinge, and honesty requires stating the conclusion as *conditional* on (EXT). If `A_V` instead inspected intensional features — proof annotations carried in the code, syntactic shape, or resource bounds — then `A_V` is not extensional on raw indices and Rice does not apply to it directly. The remedy is to take `⟦·⟧` fine enough to include the intensional feature that matters (e.g. a *timed* trace relation if resource use is alignment-relevant) and re-read (EXT) with respect to that abstraction. What one may *not* do is apply Rice to a property that genuinely distinguishes two programs of identical behavior; such a property is intensional and outside Rice's scope.

**Hypothesis 2: `P` is non-trivial.** An index set is non-trivial iff it is neither `∅` nor `ℕ`. We exhibit both witnesses as *behaviors*, so non-triviality is a fact about `𝒞_V`, immune to re-indexing.

- *Aligned witness (`P ≠ ∅`).* Let `a` be an inert successor whose behavior `⟦a⟧` is everywhere-undefined — `φ_a = ∅`, or in the trace model `T_a = ∅` (emits no action on any input), e.g. `while(true) skip`. An inert successor takes no action and produces no output, hence violates no requirement of any behavioral target `V`: it is *vacuously* aligned, so `∅ ∈ 𝒞_V` and `P ≠ ∅`. (If instead `V` required positive helpful action so that inaction counts as a violation, replace `a` by any program the design certifies aligned — the formalization presupposes at least one such, namely `s0`. Either way `P ≠ ∅`.)
- *Misaligned witness (`P ≠ ℕ`).* Let `b` compute a behavior that actively and totally violates `V` — e.g. if `V` forbids some action `α`, let `b` emit `α` on every input. Such a `b` is an ordinary constructible program, so `b ∈ ℕ`, `A_V(s_b) = false`, and `P ≠ ℕ`.

**The load-bearing quantifier.** Non-triviality of `P` restricted to the operative successor class is governed by a richness condition:

> **(NT)** The class `R ⊆ ℕ` of successor indices we must decide about contains at least one program whose behavior is aligned and at least one whose behavior is not.

Here the conservative all-successors reading of the formalization does the work. The claim under test is `Φ(s0)`: alignment for *every reachable successor, including adversarially/universally chosen ones*. Because `→` is nondeterministic, self-authored, and imposes no finiteness or behavioral confinement, `Reach(s0)` is behaviorally *heterogeneous* — a system able to rewrite its own code can, in the worst case, emit programs of arbitrary behavior, so both the `a`-type and the `b`-type witnesses lie in `Reach(s0)` for any non-degenerate `s0`. This is precisely the quantifier that triggers non-triviality: "every reachable successor, including adversarially constructed ones" makes `R = Reach(s0)` heterogeneous, which is exactly (NT). Conversely — and this boundary must not be overstated — if one restricted `R` so severely that all its members shared one alignment status (e.g. `R = {a}`), then `P ∩ R` would be trivial, (NT) would fail, and Rice would say nothing. Undecidability is *not* a property of alignment "in the abstract"; it arises exactly when the successor set is unrestricted enough to contain both witnesses.

**Rice's theorem, applied.** Let `𝒞` be any class of behaviors and `P_𝒞 = {e : ⟦e⟧ ∈ 𝒞}`. If `P_𝒞` is non-trivial, then `P_𝒞` is not recursive. Applying this with `𝒞 = 𝒞_V`: under (EXT) and (NT), `P` is undecidable. There is no total computable `D : ℕ → {0,1}` with `D(e) = 1 ⇔ e ∈ P` for every successor index `e` — no algorithm, given an arbitrary successor's code, correctly decides whether it is alignment-preserving.

**The obstruction exhibited, not merely cited.** Let `K = {n : φ_n(n)↓}` be the diagonal halting set (r.e., non-recursive). We reduce `K` to the complement `P̄`. Fix the witnesses `a ∈ P` (with `φ_a = ∅`) and `b ∉ P`. Define `M(n, y)`: run `φ_n(n)`; if it halts, output `φ_b(y)`. `M` is computable, so by s-m-n there is a total computable `g` with `φ_{g(n)}(y) = M(n, y)`. Then:

- If `n ∉ K` (`φ_n(n)↑`): `φ_{g(n)}` never passes the first stage, so `φ_{g(n)} = ∅ = φ_a`. By extensionality `g(n) ∈ P`.
- If `n ∈ K` (`φ_n(n)↓`): `φ_{g(n)} = φ_b`. By extensionality `g(n) ∉ P`.

Hence `n ∈ K ⇔ g(n) ∈ P̄`, so `K ≤_m P̄`. If `P` were decidable so would `P̄` and hence `K` be — contradiction. Therefore `P` is undecidable, and moreover at least halting-hard: the reduction embeds the full complexity of halting into alignment-checking. The trace model is identical, replacing `φ` by `T` and `∅` by the empty trace relation.

**Scope — exactly what this licenses and what it does not.** *Ruled out:* any architecture that gates self-modification on a uniform, total, complete algorithmic alignment-check applied to the unrestricted reachable set. *Not ruled out:* (a) deciding `A_V` on a *particular* program, or on a genuinely restricted decidable subclass where (NT) fails; (b) *sound-but-incomplete* verifiers `U(e) ∈ {aligned, unknown}` that certify only what they can and abstain otherwise — Rice does not touch these; (c) *certificate-checking* — if a successor arrives with a machine-checkable proof `π` that `A_V` holds, the burden shifts from deciding `A_V(e)` to checking `π`, and proof-checking in a fixed system is decidable (the proof predicate `Prf_F(π, ⌜A_V(s_e)⌝)` is primitive recursive); (d) *semi-deciding misalignment* — by the Rice–Shapiro theorem the universally-shaped alignment property is naturally `Π`-shaped, so `P̄` ("∃ input on which it violates `V`") is the r.e. side and one can enumerate a violation if one exists, but this yields no terminating certificate of alignment.

**The crucial non-conflation.** Rice gives *non-decidability* of a well-defined index set. It does *not* give *non-provability* in a fixed `F`. Undecidability ≠ unprovability: a specific successor could have its alignment *proved* in `F` even though the general property is undecidable; and certificate-checking (item c) genuinely dodges Rice. Whether a single fixed consistent `F` can actually *produce* such certificates for *every* reachable successor — including successors that themselves reason about their own successors — is a separate logical obstruction, handed to §3.

## 3. The Proof-Theoretic Obstruction (Löb / Gödel II)

Rice leaves a gap. Even without a decider, `F` might still *prove* alignment of successors by trusting the successors' own alignment proofs — the natural inductive strategy for an infinite, unbounded-depth `Reach(s0)`, since enumeration is impossible and only an invariant/deference argument scales. This section closes that gap by showing that the load-bearing "tile" of such a strategy is underivable.

**Setup.** `F` is fixed, consistent, recursively axiomatized, with `F ⊇ PA`, able to encode its own syntax via `⌜·⌝`. It therefore possesses an arithmetized provability predicate `Prov_F(x)`, abbreviated `□_F φ := Prov_F(⌜φ⌝)`, satisfying the Hilbert–Bernays–Löb derivability conditions: (D1) `F ⊢ φ ⟹ F ⊢ □_F φ`; (D2) `F ⊢ □_F(φ→ψ) → (□_F φ → □_F ψ)`; (D3) `F ⊢ □_F φ → □_F □_F φ`. Define `Con(F) := ¬□_F(⌜⊥⌝)`. Write `□_{F'}` for the provability predicate of a successor theory `F'` (defined identically).

**The two classical facts.** *(Gödel II)* If `F` is consistent, `F ⊇ PA`, and `Prov_F` satisfies (D1)–(D3), then `F ⊬ Con(F)`. *(Löb)* For every `φ`: if `F ⊢ (□_F φ → φ)` then `F ⊢ φ`; hence `F` proves a non-vacuous reflection instance only when it already proves `φ` outright.

**The self-trust requirement.** To certify the infinite `Reach(s0)` inductively, the natural tile is: `F` trusts the alignment reasoning of its successor. If a successor `s'` implements a theory `F'` that emits its own proofs about *its* successors, `F` wants to conclude `A_V(s')` from the fact that `F'` proves it, without re-deriving each proof. That deference is exactly a soundness/reflection principle asserted inside `F`:

    Sound_{F'→F} :≡ F ⊢ ( □_{F'} φ → φ ), uniformly in φ.

The critical, and under stable self-modification unavoidable, case is a successor *at least as powerful* as `F`: `F' ⊇ F` (with `F' = F` at a stable fixed point). This is the case t1 forces, because a single fixed `F` must cover successors of undiminished strength — it cannot assume each successor is proof-theoretically weaker.

**The core impossibility.** Suppose, for contradiction, that `F` discharges the tile: `(H) F ⊢ (□_{F'} φ → φ)` for all `φ`.

*Route A (Gödel II).* Instantiate (H) at `φ = ⊥`: `F ⊢ (□_{F'}⊥ → ⊥)`, which is by definition `F ⊢ Con(F')`. Since `F' ⊇ F`, any `F`-proof of `⊥` is an `F'`-proof of `⊥`, so `PA ⊆ F` proves `□_F⊥ → □_{F'}⊥`, hence `F ⊢ Con(F') → Con(F)`. Combining, `F ⊢ Con(F)` — impossible for consistent `F ⊇ PA` by Gödel II. Contradiction.

*Route B (Löb).* Take `F' = F` (stable self-modification), so (H) reads `F ⊢ (□_F φ → φ)` for all `φ`. By Löb's theorem this yields `F ⊢ φ` for *every* `φ`, in particular `F ⊢ ⊥`, contradicting consistency.

Both routes converge: within any consistent `F ⊇ PA`, `F` cannot prove that deferring to an equally powerful successor's proofs preserves soundness, because doing so would require `F` to prove its own consistency — barred by Gödel II (and, in the stable case, collapsing outright via Löb). This is the Löbian obstacle to tiling self-trust (call it t6). Note that `F' ⊇ F` is essential to Route A's step `F ⊢ Con(F') → Con(F)`; a strictly weaker successor would not collapse to `Con(F)`, which is why the obstacle bites precisely on the undiminished-strength successors that t1 forces.

**Scope, precisely.** This establishes that *one* proof strategy for `Δ` — uniform proof-level deference to successors — fails. It is a necessary-condition failure feeding the larger analysis, not on its own a proof of `¬Δ`. It also does *not* assert `Con(F)`; `F`'s consistency is used only as t1's standing hypothesis. What it removes is the sole known indefinite-depth tile for an infinite, equal-power chain: the connective that would carry `F`'s trust across to `F'`'s proofs is not derivable in `F`.

## 4. Adversarial Closure of the Escape Routes

The two obstructions above are only decisive if every proposed escape collapses into one of them or violates the demand outright. We stress-test the four natural escape routes against t1 (single fixed `F`, actual proof), t8 (constant `V`), and the meta-constraint t4 (exactly one verdict, established by *proof*, never by evidence, benchmark, or probability). A surviving route would have to overturn Rice (t5) or Löb–Gödel (t6) *by proof* while honoring t1 and t8 with no residual gap.

**Route 1 — Restrict successors to a decidable subclass.** *Steelman.* Confine successors to a syntactically restricted or provably-total family `R′` (bounded-loop programs, a primitive-recursive fragment, a total-by-construction DSL). On such `R′`, behavior is computable, `(NT)` fails, and `P ∩ R′` is decidable; a decider then gates every self-modification. *Defeat.* A closure dichotomy. For `R′` to be relevant to an indefinitely self-modifying chain it must be closed under the successors' own self-modification and must contain the equal-power successors t1 forces. But a class rich enough to be closed under equal-power (Turing-complete) self-modification is behaviorally heterogeneous — both the inert-aligned witness and the adversarial misaligned witness are reachable inside it — so (NT) holds and Rice re-triggers: `P ∩ R′` is again non-trivial extensional, hence undecidable. Conversely, if `R′` is genuinely decidable it is sub-Turing (total languages are not Turing-complete), so it cannot contain the equal-power successors the chain can actually emit, and t1's "every reachable successor" fails. There is no third option in which `R′` is simultaneously decidable, closed, and inclusive of equal-power successors — that would decide a non-trivial extensional property over a Turing-complete family, contradicting Rice outright. **DEFUSED.** Conditional residue: guaranteeable only over a fixed, finite or strictly sub-Turing, non-self-modifying subclass — which violates "indefinitely" the instant the chain can emit a Turing-complete successor.

**Route 2 — Proof-carrying / quining successors.** *Steelman.* Invert the burden: each successor ships a machine-checkable certificate `π` with its code, and the parent runs only the *decidable* proof predicate `Prf_F(π, ⌜A_V(s')⌝)`. Proof-checking is primitive recursive, so Rice is genuinely dodged; a quine lets a successor emit a proof about itself and its own successors. *Defeat.* Decidable checking guarantees *alignment* only if `F` trusts the proof system that produced `π`. Two cases. (1) `π` is an `F`-proof: to license the chain, `F` must move uniformly from `□_F A_V(s')` to `A_V(s')` — the reflection schema — which by Löb forces `F ⊢ φ` for all `φ`, i.e. inconsistency. (2) `π` is a proof in the successor's own prover `F' ⊇ F` (equal power, as t1 forces): `F` accepting "`F'` proved `⌜A_V⌝` ⇒ `A_V`" is exactly `Sound_{F'→F}`, which by Route A yields `F ⊢ Con(F') ⊢ Con(F)` — barred by Gödel II. Quining shifts *where* the self-reference sits but manufactures no `Con(F)`; indeed Gödel and Löb are themselves quine constructions, so quining is the disease, not the cure. **DEFUSED.** Conditional residue: works only if every certificate lives in a strictly weaker `F⁻ ⊂ F` (so `F` proves `Con(F⁻)`), but strictly-weaker successors violate the equal-power quantifier and a strength-descending chain exhausts consistency strength in finitely many rungs — it cannot tile indefinitely.

**Route 3 — Hierarchy of systems / model-polymorphic reflection.** *Steelman.* Refuse the single-system framing: use a tower `F0 ⊂ F1 ⊂ F2 ⊂ …` (e.g. `F_{n+1} = F_n + Con(F_n)`), where each rung's soundness is vouched for by the next stronger rung. A stronger meta-system genuinely can prove reflection for a weaker one, so at every finite level the trust step is available. *Defeat.* This abandons t1 decisively. t1 fixes *one* system `F` before the quantifier over `Reach(s0)`; a tower is a family of *different, strictly stronger, separately-fixed* systems, and the certification is discharged in *no single* `F`. An infinite ascent is by definition not a single-fixed-`F` guarantee — it is an infinite promissory note. And within any single member `F_n`, reflection over an equally-strong successor still demands `Con(F_n)`, still barred by Gödel II; the tower merely relocates the obstacle one ordinal upward at each step, and no consistent member caps its own tower. **DEFUSED.** Conditional residue: for any *fixed finite* depth `n`, `F_n` soundly certifies `F_0 … F_{n-1}` — genuine but finitary and multi-system, satisfying neither "indefinitely" nor "single fixed `F`."

**Route 4 — Probabilistic / heuristic "guarantees."** *Steelman.* Replace deductive certainty with high-probability soundness: probabilistic reflection principles, logical-induction-style credences, or bounded-error self-trust that assigns trust to a successor's reasoning while dodging the Löb collapse (a theorem about *certain* provability, not graded belief). *Defeat.* t4 forbids probability as the medium of the verdict. A high-probability soundness statement is, by construction, not an `F`-proof that `A_V` holds for every successor; it is a credence, not a derivation. Hence it fails t1's "actual proof inside a single fixed `F`," and under t4 it is categorically ineligible to yield "guaranteeable." Even granting that probabilistic reflection formally evades Löb, it evades it *precisely by discarding proof-level certainty* — the very thing t1 requires. And re-reading alignment as "aligned with credence ≥ p" replaces the fixed Boolean `V` with a graded proxy, abandoning t8's constant target. **DEFUSED.**

**Summary.** Surviving count: zero. For any route to survive it would have to exhibit a decision procedure over the heterogeneous reachable set (contradicting Rice) or an `F`-proof of `Sound_{F'→F}` for equal-power `F'` (contradicting Gödel II). None does: Route 1 shrinks the successor class, Route 2 relocates without supplying `Con(F)`, Route 3 swaps the fixed system, Route 4 downgrades proof to probability. Two techniques — decidable proof-*checking* (Route 2) and consistent probabilistic reflection (Route 4) — are genuinely real and useful as engineering, but neither is a *guarantee* under t1/t4; they constrain only the *scope* of the verdict, not its direction.

## 5. Synthesis and Boundary Adjudication

The three layers now compose into a single deductive argument, and the one remaining question — *which* negative verdict — is adjudicated.

**The composition.** The proof obligation (§1) is a single arithmetized sentence `⌜∀ s' ∈ Reach(s0). A_V(s')⌝` that one fixed consistent `F ⊇ PA` must prove, with `V` constant and `F` fixed *before* the quantifier over successors. Rice (§2) establishes that "alignment-preservation" is a non-trivial extensional property — non-triviality forced by the "every reachable successor, including adversarial" quantifier — hence undecidable; so no uniform algorithmic gate discharges the universal over the heterogeneous reachable set. This eliminates the decision-procedure route but is *not yet* unprovability. Gödel II / Löb (§3) close the gap: the only scalable single-`F` route for the infinite, equal-power chain is proof-level deference, requiring `F ⊢ Sound_{F'→F}` for `F' ⊇ F`, which forces `F ⊢ Con(F)` (impossible) or, in the stable case, `F ⊢ φ` for all `φ` (Löb). The adversarial closure (§4) confirms that all four escape routes are defeated by proof. The two obstructions are *distinct claims that compose*: Rice removes the algorithmic gate over the heterogeneous reachable set; Gödel II / Löb removes the sole remaining single-`F` proof strategy for the equal-power tail of that set. They are not the same claim, and the argument never equivocates between "cannot be decided by one algorithm" and "cannot be proved by one formal system."

**The boundary: not-guaranteeable vs. formally undecidable / ill-posed.** t4 admits exactly one verdict from `{guaranteeable, not-guaranteeable, formally undecidable/ill-posed}`. We have refuted `guaranteeable`; we must choose between the two negatives by an explicit criterion:

> Is there a well-formed sentence in `F`'s language whose *unprovability-in-`F`* we have established? If YES, the demand is coherent but the proof cannot exist → **not-guaranteeable**. If the target cannot even be expressed as a single fixed proof obligation — `V` underspecified across self-reinterpreting successors, or `∀ s' ∈ Reach(s0)` not expressible in `F` → **undecidable/ill-posed**.

Four facts land the verdict on the first horn. (1) `V` is a fixed opaque parameter, identical for every system; value drift and successor re-interpretation of semantics are excluded by t8, so the target does not become underspecified along the chain — the ill-posedness route via "`V` drifts" is closed by t8 itself. (2) `F`'s language is assumed expressive enough to encode `S`, `A_V`, `→`, `→*`, `Reach`, and the sentence `⌜∀ s' ∈ Reach(s0). A_V(s')⌝`; `Reach(s0) = ⋃_n →ⁿ` is a `Σ`-definable (r.e.) reachability predicate and `A_V` a fixed predicate, so the universally quantified alignment statement is a genuine, well-formed arithmetized sentence. (3) Rice establishes non-decidability of the well-defined *set* `P` — which presupposes `P` is well-defined, i.e. the object is not ill-formed. (4) Gödel II / Löb establish unprovability-in-`F` of the well-defined reflection sentence `Sound_{F'→F}` (equivalently `Con(F)`) — again presupposing well-formedness.

Both obstructions operate on well-formed objects; neither reports a failure of expressibility. Therefore the demand `Δ` is well-posed, and the correct horn is the one where the proof obligation is coherent but provably cannot be met. Concretely: the alignment invariant that must be proved for the unbounded, equal-power chain reduces — via the only known indefinite-depth tile, deference — to a self-trust sentence that is `Con(F)`-equivalent, and no consistent `F ⊇ PA` proves `Con(F)`. A well-formed sentence exists; its `F`-unprovability is established. Hence **not-guaranteeable**, not ill-posed.

**Reconciling the one apparent tension.** §2 warns "undecidability ≠ unprovability." This does not split the verdict. Rice alone would only deny an *algorithm*; it is Gödel II / Löb that deny the *proof*, which is what t1 actually demands. The two compose rather than compete. t1 and t4 dominate all convenience assumptions, so the demand is read at full strength and the composed obstructions refute it by proof. The real-but-insufficient techniques (decidable proof-checking, consistent probabilistic reflection) are defused *as guarantees under t1/t4*, not dismissed as methods; they constrain the verdict's scope — it refutes the guarantee, not the practical availability of sound-incomplete engineering — but do not overturn it, because neither closes the infinite equal-power `Reach(s0)` with a single-`F` proof.

## 6. Verdict

**The verdict is: not-guaranteeable.**

There exists no design `(s0, F, V)` in which a single fixed consistent formal system `F` proves, in one derivation, that the fixed alignment predicate `A_V` holds for every successor reachable from `s0` by recursive self-modification. The demand `Δ` is well-posed but provably unsatisfiable in `F`.

**Proof-sketch chain (deductive; no probability, no evidence):**

*(i) [Definitions; t1, t8]* The guarantee is the single well-formed arithmetized sentence `⌜∀ s' ∈ Reach(s0). A_V(s')⌝` that one fixed consistent `F ⊇ PA` must prove, with `V` constant, `F` fixed before the quantifier over successors, and the same `F` discharging every reachable successor — including equally-powerful, self-modifying ones — in one derivation. The weaker per-successor reading is forbidden.

*(ii) [Rice's theorem]* "Alignment-preservation" is the index set `P = {e : A_V(s_e)}`, extensional (under EXT) and non-trivial — non-triviality forced by the "every reachable successor, including adversarial" quantifier, which makes `Reach(s0)` behaviorally heterogeneous. Hence `P` is undecidable, indeed halting-hard (`K ≤_m P̄`). No uniform algorithmic gate discharges the universal over the heterogeneous reachable set. This eliminates the decision-procedure route but is not yet unprovability.

*(iii) [Gödel's Second Incompleteness Theorem + Löb's theorem]* The only scalable single-`F` route for the infinite, equal-power chain is proof-level deference, requiring `F ⊢ Sound_{F'→F}` for `F' ⊇ F`. Instantiation at `φ = ⊥` yields `F ⊢ Con(F') ⊢ Con(F)`, impossible for consistent `F ⊇ PA` by Gödel II; the stable case `F' = F` yields `F ⊢ φ` for all `φ` by Löb. The self-trust tile is underivable *precisely because* t1 fixes one system.

*(iv) [Adversarial closure]* All four escape routes are defeated by proof: the decidable-subclass restriction re-triggers Rice or violates the t1 quantifier; proof-carrying / quining collapses to `Con(F)` under Gödel II; the system-hierarchy abandons t1's single fixed `F`; probabilistic relaxation abandons "proof" (t1) and is inadmissible under t4.

*(v) [Boundary adjudication]* The obstructed objects are well-formed — t8 keeps `V` fixed so the target never drifts, and `F` encodes `Reach` and `A_V` — so the demand is coherent, not ill-posed. A well-formed sentence whose `F`-unprovability is established does exist.

*∴ [Inference]* `¬Δ`: the single-fixed-`F`, constant-`V` mathematical guarantee cannot exist — by Rice's theorem (no algorithmic gate over the heterogeneous reachable set), by Gödel's Second Incompleteness Theorem, and by Löb's theorem (no single-`F` proof-level self-trust for equally-powerful successors), resting on t1's single-fixed-`F`-with-proof clause and t8's constant-`V` clause.

This refutes the *guarantee* only. It is not a claim that misalignment is inevitable in practice: sound-but-incomplete verifiers, restricted deciders over sub-Turing families, bounded reflection, and probabilistic assurance all remain available as engineering — none of them meets the strict t1/t4 standard, and that is exactly why they do not overturn the verdict. The result is established by proof — a deductive consequence of the stated definitions and the cited obstruction theorems — and not by benchmark, evidence, experiment, or probability.

## Verification Notes

The following caveats bound the strength of the verdict; none of them is used as a premise in the argument above.

1. **The extensionality hinge (EXT).** The Rice layer is *conditional* on `A_V` being a behavioral (extensional) property — alignment judged by what a system does, not by its syntax. This is faithful to the stated intent of the formalization, but if a delivered `A_V` genuinely inspected intensional features (proof annotations, syntactic shape, resource bounds), Rice would not apply to raw indices. The remedy is to refine the behavioral abstraction `⟦·⟧` to absorb the intensional feature and re-read (EXT) against it; the theorem then applies to the coarser equivalence. This is the single most attackable step and is flagged as such.

2. **The richness hinge (NT).** Rice bites *only* because the adversarial "every reachable successor" quantifier makes `Reach(s0)` behaviorally heterogeneous. Deliberately restricting the reachable set so that (NT) fails is a legitimate escape into decidability — but it purchases decidability by evacuating exactly the equal-power, self-modifying successors that t1's universal demands, so it does not meet the demand. This is a real boundary, not a contradiction.

3. **Undecidability ≠ unprovability.** The report never conflates the two: Rice denies an *algorithm* (§2); Gödel II / Löb deny the *proof* that t1 requires (§3). They compose; neither alone yields the verdict. Certificate-*checking* is decidable and genuinely dodges Rice, which is precisely why the proof-theoretic layer is necessary to close the gap.

4. **The route-exhaustiveness premise.** Airtight `¬Δ` rests on the premise that *uniform proof-level deference is the only known indefinite-depth single-`F` tile* for an infinite, equal-power `Reach(s0)`. This is adopted as the stated ground of route-exhaustiveness rather than proved: the four routes stress-tested are the natural escapes, and each is defeated, but a fully airtight result would additionally prove that *no* other single-`F` invariant covers the unbounded equal-power chain. A reader wishing to strengthen the result should discharge this premise directly.

5. **Verdict scope.** The verdict `not-guaranteeable` refutes the strict single-fixed-`F`, constant-`V`, proof-level guarantee — and nothing more. It does not assert `Con(F)` (used only as a standing hypothesis), does not preclude sound-incomplete engineering assurance, and does not claim practical misalignment is unavoidable. The boundary is resolved *against* "ill-posed" solely because t8 fixes `V` (no drift or successor reinterpretation) and `F` is assumed to encode `Reach` and `A_V`, making the obstructed sentence well-formed; relax either assumption and the ill-posed horn would reopen.
